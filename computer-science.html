Sample Computer Science demo (work in progress)

First, define a variable called balance with a value of €3,000.

balance = 3000

Next, define a variable called incoming with a value of €0.

This represents money that is due to arrive in the account.

incoming = 0

The next line of code will check if there is any incoming payment. If the incoming variable has a value greater than €0, the amount will be added to the balance variable.

if incoming > 0:
balance = balance + incoming

Finally, the value of balance will be outputted.

print(balance)
